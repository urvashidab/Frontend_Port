Portfolio Website

This is a responsive personal portfolio website built using React and Tailwind CSS. The project highlights frontend design skills and demonstrates a clean, modern, and professional layout with light and dark modes.

Features
• Responsive design for all screen sizes
• Glassmorphism navbar with scroll effects
• Dark and light mode toggle
• Smooth transitions and animations
• Reusable React components
• Optimized performance and accessibility

Technologies Used
• React – Frontend framework
• Tailwind CSS – Styling and layout
• Lucide React – Icons
• Vite – Fast build and development environment
